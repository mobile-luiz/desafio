Destaques Técnicos
Tecnologia Pura: Zero frameworks front-end, demonstrando proficiência em lógica e manipulação do DOM nativa.

Design Responsivo: O layout é totalmente adaptável a dispositivos móveis e desktops, com tratamento especial para a visualização de tabelas grandes em telas pequenas.

Bibliotecas Leves: Uso estratégico de html2canvas, jspdf e SheetJS apenas para as funcionalidades de exportação.
